# Quiz-System

## Project Description

This project is a command-line based quiz application developed in Python. It allows users to take quizzes based on available questions stored in a JSON file. The system also includes an admin mode where users can add new questions.

The application demonstrates key programming concepts such as file handling, data structures, and modular programming. Future updates will include Object-Oriented Programming, decorators, timers, and statistics.

---

## Features

* Start quiz with multiple-choice questions
* Add new questions (Admin mode)
* Questions stored in JSON file
* Score calculation at the end of the quiz
* Input validation for user answers

---

## Technologies Used

* Python
* JSON (for storing questions)
* Standard Python libraries (json)

---

## Project Structure

* main.py – main program logic
* questions.json – question database
